
# EMPLOYEE RECORD

It creates full stack application for employee record.

## TECH USED

**Frontend:** JSP

**Framework:** Spring MVC

**ORM Framework:** Hibernate

**Database:** MySQL

## DEPENDENCIES

This project uses the following dependencies from maven repository,

javax.servlet-api

Version: 3.1.0

jsp-api

Version: 2.1

jstl

Version: 1.2

spring-context

Version: 5.3.16

spring-webmvc

Version: 5.3.18

Hibernate Core Relocation

Version: 5.4.32

MySQL Connector/J

Version: 8.0.16
